## 🐛 既知の問題と制限事項

### 実装済み機能

1. **対応ソース**: Markdownドキュメント
   - テーブル形式のMarkdownを自動パース
   - データを抽出してExcelに変換

2. **対応体裁**:
   - サンプルExcelからの体裁完全再現
   - デフォルト体裁での生成
   - 列幅、行の高さ、フォント、色、配置、罫線の完全適用

3. **スタイル変換**:
   - A型（ビジネス文書）: 敬語表現
   - B型（技術仕様書）: 簡潔表現、構造化
   - H型（ハイブリッド）: 敬語+構造化、空行削除

### 制限事項

1. **複雑なレイアウト**: セル結合等の複雑なレイアウトは将来対応予定

2. **コードベース直接生成**: 現時点ではMarkdownを経由

### トラブルシューティング

**問題**: Skillが起動しない

- **確認**: Claude Code を再起動したか？
- **確認**: リクエストに「Excel」などのトリガー語句が含まれているか？

**問題**: openpyxl エラー

```bash
pip install openpyxl
```

**問題**: 生成されたExcelの体裁がおかしい

- サンプルExcelのパスが正しいか確認
- デフォルト体裁を使用する
